#Rapport My Social API1

#Description
- Projet d'API REST développé avec Node.Js et MongoDB et Atlas, simulant un mini réseau social.  

L’appli permet de gerer des albums et des photos : création, lecture, modif et suppression.

#Techno utilisées
- Node.js  
- Express  
- MongoDB Atlas  
- Mongoose  
- Postman

#Installation et run
```bash
npm install
node index.mjs
